imgNotes Plugin - note display on fixed image size

This example demonstrates display on a plain fixed size image. Zoom in and out using the mousewheel. Left mouseclick and drag to pan. On touch enabled devices pinch gestures can be used to zoom in and out and tap and drag to pan around.

imgNotes Plugin - note display on dynamic size image

This example demonstrates note display on a dynamic size image. Zoom in and out using the mousewheel. Left mouseclick and drag to pan. On touch enabled devices pinch gestures can be used to zoom in and out and tap and drag to pan around.
